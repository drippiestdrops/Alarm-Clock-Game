# Alarm-Clock-Game

“ALARM CLOCK: Tick’s Timebomb” is a psychological horror Choose Your Own Adventure (CYOA) type game inspired by the unsettling feeling of waking up to an alarm clock. In this game, your alarm clock (Tick) comes to life and invokes an amount of fear into you, increasing everyday for 10 days.  However, halfway through this psychological journey, you suddenly catch feelings for the alarm clock, furthering the relationship between it and the player. Your choices each day determine what happens during the boss fight, and the outcomes after it.

Brainstormed about what everyday routine/object we were going to center our game around. We eventually decided on an alarm clock and its functions. 

Some challenges we faced when developing this game are balancing comedy and storytelling, while maintaining a fun and enjoyable experience. Coding the User Interface was also a challenge as there were many moving aspects involved. We overcame the first challenge by properly planning what dialogues to add to make the gameplay more fun and engaging, while giving the user a fun story while playing. For the second challenge, we split up each day and developed the UI for each day. For the last day, a new UI had to be implemented as a combat system was introduced.
